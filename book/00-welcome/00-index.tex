\section{Your first code}

You do not need to install anything yet. Open this chapter's notebook in
Google Colab and run your first line of Python right now.

\begin{codebox}{print()}
\begin{verbatim}
print("Welcome, Little Genius! You just ran your first line of Python.")
\end{verbatim}
\end{codebox}

\begin{knowledgecheck}{Check My Knowledge}
\begin{enumerate}
  \item What word did Python print when you ran the code above?
  \item What happens if you change the text inside \texttt{print(...)}?
\end{enumerate}
There is no wrong guess here. Try it and see what happens -- that is how you
learn it, not yet how you fail it.
\end{knowledgecheck}

Good stopping point! Stretch, take a break, and come back whenever you are
ready for Chapter 1.
