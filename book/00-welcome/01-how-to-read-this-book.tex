\section{How to read this book}

Every chapter has two tracks that sit side by side:

\begin{itemize}
  \item \textbf{Track A -- Foundations.} Short Python lessons. Each one has
    its own notebook you can run in Google Colab, right in your browser, no
    install needed.
  \item \textbf{Track B -- Build.} One game you build across the whole
    book. Chapters 0 to 7 work anywhere, Colab included. Chapter 8 is the
    first time you need to install Python on your own computer, because
    that is when the game gets a real window you can see and play.
\end{itemize}

You also have a second no-install option: the \textbf{Little Genius Zone}
(GitHub Codespaces), a full coding environment in your browser. It also
holds optional bonus problems for anyone who finishes a chapter early and
wants more of a challenge.

Not everyone needs to read every chapter in the same order. See
\texttt{learning-paths.tex} for a few different paths through the book,
depending on what you already know.
