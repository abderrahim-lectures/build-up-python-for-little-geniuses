\section{How we teach}

This section is for kids and grown-ups both.

We use a method that starts with something you can touch or picture, and
only then moves to code:

\begin{enumerate}
  \item \textbf{Concrete.} A real example, like counting blocks.
  \item \textbf{Pictorial.} A picture of the same idea, like a diagram.
  \item \textbf{Abstract.} The actual Python code that does it.
\end{enumerate}

Each chapter also opens with a few guided questions before we explain
anything -- try to answer them yourself first. Mistakes are part of how
this works. We say ``not yet,'' never ``wrong.''

Every chapter ends with two short checks: a self-graded ``Check My
Knowledge'' quiz, and a challenge that mixes in ideas from earlier
chapters, so you keep old skills sharp while you learn new ones.
