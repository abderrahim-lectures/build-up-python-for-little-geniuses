# The Little Genius Zone: bonus problems

This folder holds the optional, open-ended "unsolved problems" mentioned in
each chapter -- one or two per chapter, for kids who finish early and want
more. They have no single correct answer, they're not autograded, and they
never gate progress to the next chapter. They live here (not in
`foundations/` or `build_game/`) so the graded Track A/B code stays clean
and separate from open-ended exploration.

Files are added starting with Chapter 1 (Chapter 0 is pure orientation, no
bonus problem yet).
